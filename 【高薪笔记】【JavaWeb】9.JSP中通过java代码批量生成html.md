# 【Java Web核心开发】9.JSP中通过java代码批量生成html

```jsp
<%@ page language="java" contentType="text/html; charset=ISO-8859-1"
    pageEncoding="ISO-8859-1"%>
<!DOCTYPE html>
<html>
<head>
<meta charset="ISO-8859-1">
<title>Insert title here</title>
</head>
<body>

<% for(int i =0; i<10; i++)  { %>

<p>This is a HTML generated by for Loop. Number Count is : <%= i %> </p>

<%  }  %>


</body>
</html>
```
